Outcomes B2 · 3rd edition · lexical approach

Lexical Revision System

16 units — each with a Student page (live 120-min lesson) and a Teacher page (answer keys, methodology notes, rubrics). Progress, submissions and Leitner reviews are stored per-student in Supabase.

Method in one paragraph

Chunks-first, grammar-as-a-servant (Dellar & Walkley, Lewis, Selivan). Core chunks — direct instruction and drill. Extension chunks — met in context, not glossed unless clicked. Tasks start at Bloom's Apply and climb to Create: reformulations, roleplays, emoji-driven reviews, dubbing labs, CYOA, gallery walks, streaming-profile psy-tests. Spaced repetition on Leitner (5 boxes). Post-lesson AI reads the transcript + submissions and drafts progress updates for the teacher to confirm.

Reference

Method core: Dellar & Walkley — Teaching Lexically; Lewis — The Lexical Approach, Teaching Collocation; Selivan — Lexical Grammar; Nation — Learning Vocabulary in Another Language.

Task genres inspired by TB: Comm Activity 1.1 (snowflake cards), 1.2 (art exhibition), 2.1 (psy-test), 16.1 (elevator pitch), 16.2 (CYOA).
